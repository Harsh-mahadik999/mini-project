# College Internal Online Voting System

A web-based voting system for internal college elections. Students can log in and vote once per position, while admins manage candidates, voting settings, and live results from a dashboard.

## Features

- Student login and voting
- Admin login and dashboard
- One vote per student per position
- Live results and participation stats
- Voting start/stop control

## Tech Stack

- Frontend: HTML, Tailwind CSS, JavaScript
- Backend: Node.js, Express
- Auth: JWT

## Project Structure

```text
client/   Frontend pages
server/   Backend API and logic
server.js Main server entry
```

## Run Locally

```bash
npm install
cd server && npm install
cd ..
node server.js
```

Open: http://localhost:3000
